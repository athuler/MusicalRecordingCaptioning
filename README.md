# Musical Recording Captioning

Caption recordings of musical theater, concerts, or any video that has lyrics on Genius.

## Details

This Python CLI app uses the Genius API to download the lyrics of an album and then matches them to the provided video/audio recording to create SRT captions.

Genius API credentials can be stored in a local `.env` or passed as arguments to the command.
